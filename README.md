# Customer Purchase Prediction

This project predicts customer purchase behavior using Machine Learning.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## Features
- Data preprocessing
- Feature engineering
- Model building (Logistic Regression, Decision Tree)
- Model evaluation (Accuracy, Precision, Recall, F1-score)

## Outcome
Improved prediction performance using cross-validation and hyperparameter tuning.
